Command Summary


  • add (a): Adds a contact to ContactHub.
    Format: add n/NAME p/PHONE... e/EMAIL... a/ADDRESS b/BIRTHDAY [m/MODULE]...
  • edit (e): Edits the details of the contact identified by the index number used in the last contact listing. Existing values will be overwritten by the input values.
    Format: edit INDEX [n/NAME] [p/PHONE]... [e/EMAIL]... [a/ADDRESS] [b/BIRTHDAY] [m/MODULE]...
  • find (f): Finds contacts whose names contain any of the given keywords.
    Format: find KEYWORD [MORE_KEYWORDS]...
  • findmodule (fm): Finds modules whose names contain any of the given keywords.
    Format: findmodule MODULE_NAME [MORE_MODULE_NAMES]...
  • getmodule (gm): Shows NUSMods module information in browser.
    Format: getmodule MODULE_NAME
  • listmodules (lm): Shows a list of all modules in ContactHub.
  • delete (d): Deletes the specified contact from ContactHub.
    Format: delete INDEX
  • select (sl): Selects the contact identified by the index number used in the last contact listing.
    Format: select INDEX
  • addphoto (ap): Adds a photo to a contact using the index of the contact in the latest listing.
    Format: addphoto INDEX u/IMAGE_URL
  • addtask (at): Adds a task to the task list.
    Format: addtask t/APPOINTMENT d/DATE s/START_TIME
  • edittask (et): Edits the details of the task identified by the index number.
    Format: edittask INDEX [t/APPOINTMENT] [d/DATE] [s/STARTTIME]
  • deletetask (dt): Deletes the task identified by the index number in the task list.
    Format: deletetask INDEX
  • summary (sm): Opens the ContactHub Command Summary.
  • venue (v): Shows NUSMods venue information.
  • bus (b): Opens the NUS bus routes.
  • map (m): Opens the NUS map.
  • help (hp): Opens the User Guide.
  • history (h): Shows the commands that were previously entered.
  • list (l): Shows a list of all contacts.
  • clear (c): Clears ALL contact entries.
  • undo (u): Restores ContactHub to the state before the previous undoable command was executed.
  • redo (r): Reverses the most recent undo command.
  • exit (x): Exits the program.

Keyboard Shortcuts


  • Escape: Clears the whole command box.
  • Alt: Relocates text cursor to the right completely.
  • Delete: Deletes a chunk of words or chunk of whitespace on the left of the text cursor.
  • Insert: Fills the command box with the AddCommand parameters.
